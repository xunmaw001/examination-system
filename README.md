# examination-system
基于Spring Boot和Vue.js的考试系统是一款高效的在线评测平台。前端采用Vue.js构建，提供直观的用户界面，支持题库管理、在线作答和成绩查询。后端使用Spring Boot，确保数据处理安全稳定，支持用户注册、试卷生成和成绩统计。该系统旨在提升考试过程的便捷性和透明度，帮助教师和学生高效管理考试与评估，提高学习效果。
